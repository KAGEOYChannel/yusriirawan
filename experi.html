Pengenalan Excel

Microsoft Excel adalah program pengolah angka untuk membuat tabel, menghitung data, dan menampilkan grafik. Excel sering digunakan untuk mencatat nilai, data keuangan, atau perhitungan otomatis menggunakan rumus.

Bagian-bagian Utama Excel

  • Title Bar: Menampilkan nama file Excel.
  • Ribbon: Tempat tab menu dan ikon perintah.
  • Worksheet Area: Tempat kita bekerja, berisi kolom (A, B, C...) dan baris (1, 2, 3...).
  • Formula Bar: Tempat mengetik rumus atau melihat hasil perhitungan.

Catatan Penting

  • Gunakan tanda sama dengan (=) untuk menulis rumus di Excel.
  • Contoh: =SUM(A1:A4) untuk menjumlahkan data dari sel A1 sampai A4.
  • Rumus lain: =AVERAGE(), =MAX(), =MIN().
